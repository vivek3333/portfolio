Parking Space Finder

Parking Space Finder is a computer vision-based application designed to detect available parking slots in real time using video input. By capturing the top view of a parking lot through a camera, the system processes the feed using libraries like OpenCV and NumPy. It identifies each parking space's coordinates and applies image processing techniques to determine whether a car is present or not. The result is visually displayed on-screen, showing available and occupied spots.
